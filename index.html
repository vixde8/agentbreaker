    max_tokens = 4096
    is not enough.

    That cap covers a single call. Your agent makes 50. Each one looks fine. Together, they burn $12 before anyone notices.

    Observability tools log what happened. They don't stop it. AgentBreaker does.

    agent.py without AgentBreaker 
    # iteration 1 — $0.008
result = llm.invoke("research quantum computing")
# iteration 2 — $0.008  (same query)
result = llm.invoke("research quantum computing")
# iteration 3 — $0.008  (still stuck)
result = llm.invoke("research quantum computing")
# ... 47 more calls. Total: $0.41. Agent still running.
    agent.py with AgentBreaker 
    # iteration 1 — $0.008 ✓
result = llm.invoke("research quantum computing")
# iteration 2 — $0.008 ⚠ same args flagged
result = llm.invoke("research quantum computing")
# 🔴 KILL — stuck_loop: 2 identical calls. Saved: ~$0.39
# Agent stopped. Run logged. Budget intact.
    // Guardrails

    Pick the rules that fit your agent.

    All rules are opt-in. Configure per run from the dashboard or via code.

    Rule Triggers when Action
    stuck_loop Same tool called with identical args N times 🔴 KILL
    cost_exceeded Total spend exceeds your dollar budget 🔴 KILL
    velocity_exceeded Spend rate spikes in a rolling time window 🔴 KILL
    iterations_exceeded Iteration count hits your configured cap 🔴 KILL
    cost_spike Single call costs 5× the session average 🔴 KILL
    output_bloat Single response exceeds token threshold ⚠ WARN
    time_exceeded Run wall time exceeds your time limit 🔴 KILL
    // Integration

    Two lines.
    Any framework.

    Native adapters for LangChain and the OpenAI Agents SDK. No monkey-patching. No code rewrites. Import, configure, go.

    🦜 LangChain → Callback Handler
    🤖 OpenAI Agents SDK → RunHooks
    ⚡ Groq / OpenAI / Anthropic → Any LLM
    Read integration docs →
    LangChain OpenAI Agents 
    from agentbreaker import LangChainCircuitBreakerCallback
from langchain_groq import ChatGroq

# Configure once
breaker = LangChainCircuitBreakerCallback(
    run_id="my-run-001",
    config={
        "max_cost_usd": 0.50,
        "max_iterations": 10,
        "rules": ["stuck_loop", "cost_exceeded"],
    }
)

# Attach to any LangChain LLM
llm = ChatGroq(
    model="llama-3.3-70b-versatile",
    callbacks=[breaker]   # ← that's it
)

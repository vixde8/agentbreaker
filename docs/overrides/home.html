AgentBreaker is an inline circuit breaker for LLM agents. It detects stuck loops, token bloat, and cost spikes in real time — and stops the agent before it's too late.

Why AgentBreaker

Observability tools watch. AgentBreaker stops.

LangSmith and Langfuse tell you an agent went wrong after the fact. AgentBreaker kills it while it's happening.

🔁

Argument-Aware Loop Detection

Detects when an agent calls the same tool with identical arguments repeatedly — a true stuck loop, not legitimate deep research.

💸

Real-Time Cost Limits

Set a dollar budget per run. The breaker hard-stops the agent the instant that limit is crossed — not after the API call is billed.

📊

Token Bloat Detection

Catches when a single LLM response is excessively verbose. A question like "Who is the PM of India?" shouldn't return 450 tokens.

⚡

Spend Velocity Guard

Monitors how fast money is being spent in a rolling time window. Catches sudden token-rate explosions before they compound.

🎛️

Composable Rule Engine

Pick exactly which rules apply per run. Mix and match: cost limit + loop detection + time cap, all configurable from the UI.

📈

Live Dashboard

Real-time cost trajectory chart, per-call token bars, run history, and side-by-side comparison mode showing cost savings.

🔌

LangChain Native

Drop in LangChainCircuitBreakerCallback — no monkey-patching, no code rewrites. Uses LangChain's official callback API.

🐳

Zero-Config Docker Setup

One command to run the full stack: backend API, React dashboard, SQLite database. No environment setup headaches.

🆚

Side-by-Side Comparison

Run the same query with and without the breaker. See the actual cost delta and token savings. Your proof-of-value screenshot.

Quick Start

Up and running in 5 minutes

1

Clone the repo

git clone https://github.com/vixde8/agentbreaker.git && cd agentbreaker

2

Get a free Groq API key

No credit card. Visit groq.com → sign up → copy your key.

3

Set your key

echo "GROQ_API_KEY=your_key_here" > .env

4

Run it

docker compose up --build → open localhost:3000
